# Begin your contrubution

## Step 1: Fork project

Fork https://gitlab.com/pc-magas/asciiart-html-theme into your own repository


## Step 2: Checkout dev

```
git checkout dev
```

## Step 3: Checkout a new feature branch

```
git checkout my feature
```

## Step 5: Run project localy

With this command:

```
npm start
```

1. You build SASS
2. Minify JS
3. Build nunjunks templates for website
3. Enable Hot reload 

Upon your terminal you'll be prompted upon a browser session use this to see your result.

## Step 6: PR

Once your feature is ready make a PR into https://gitlab.com/pc-magas/asciiart-html-theme
