Navbar

Basic usage

Use for main navigation

Item 1 Item 2 Item 3

Code:

            

                
HTML

                
            

            
                <div class="nav-wrapper">
                    <nav class="main-nav">
                        <a href="#">Item 1</a>
                        <a href="#">Item 2</a>
                        <a href="#">Item 3</a>
                    </nav>
                </div>

Sticky Navbar

If you want the navbar to stick on top of thew screen use sticky-top class:

HTML
                
                    <div class="nav-wrapper sticky-top">
                        <nav class="main-nav">
                            <a href="#">Item 1</a>
                            <a href="#">Item 2</a>
                            <a href="#">Item 3</a>
                        </nav>
                    </div>

Navbar controls

You can use lead class in order to place buttons or icons in front of main navigation:

*
Item 1 Item 2 Item 3
HTML
                
                    <div class="nav-wrapper sticky-top">
                        <div class="lead">
                            <button>*</button>
                        </div>
                        <nav class="main-nav">
                            <a href="#">Item 1</a>
                            <a href="#">Item 2</a>
                            <a href="#">Item 3</a>
                        </nav>
                    </div>

CodeWrapper

If you want to place a piece of code into your documentation use:

HTML
            
                <div class="code-wrapper">
    <div>
        <h5>HTML</h5>
        <button class="copy">COPY</button>
    </div>
    <pre>
        <code>
            Piece of coide here            
        </code>
    </pre>
</div>

Once initialized then you also need to initialize the copy buttons:

JS
            
                <script type="application/javascript" src="/js/asciiart.min.js></script>
                <script>
                document.addEventListener("DOMContentLoaded",function(){
                    AsciiartTheme.autoinit.codewrapper()
                })
                </script>

In case of using Modules

JS
            
                <script type=\"module\">
                import { autoinit } from "/js/asciiart.esm.js"; 
                document.addEventListener("DOMContentLoaded",function(){
                    autoinit.codewrapper()
                })
                </script>

Otherwise you can use the CodeWrapper class:

JS
            
                <script type=\"module\">
                import { CodeWrapper } from "/js/asciiart.esm.js"; 
                document.addEventListener("DOMContentLoaded",function(){
                    new CodeWrapper("#mycontainer")
                })
                </script>

JS
            
                <script type="application/javascript" src="/js/asciiart.min.js></script>
                <script>
                document.addEventListener("DOMContentLoaded",function(){
                    new AsciiartTheme.CodeWrapper("#mycontainer")
                })
                </script>
