AsciiArt Theme

A lightweight retro-inspired HTML theme designed to feel like the early web. No frameworks. No JavaScript required. Just semantic HTML and clean CSS.

It focuses on readability, simplicity, and nostalgic terminal aesthetics.

Philosophy

The modern web is often heavy, cluttered, and slow. This theme embraces a different idea: pages that load instantly, remain readable everywhere, and feel like a personal homepage from the early internet.

  • Simple layout
  • Semantic HTML
  • Fast loading
  • Accessible typography
  • No dependencies
