TAR(1)

NAME

tar - archive files

SYNOPSIS

tar [OPTION...] [FILE...]

DESCRIPTION

tar stores and extracts files from an archive.

OPTIONS

-f ARCHIVE, --file=ARCHIVE
Use archive file ARCHIVE.
-C DIR, --directory=DIR
Change to directory DIR.
-v, --verbose
Verbose output.

EXAMPLES

tar -cf archive.tar file1
tar -xf archive.tar
