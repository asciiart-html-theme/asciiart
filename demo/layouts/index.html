About

This theme supports 3 layouts:

  • Sidebar with header upon aside (aka Sidebar Full)
  • Sidebar upon content
  • No sidebar

Sidebar Full

This current page is a prime example of this layout. Sidebar is on the left side of the page taking over the full height of the page
