About

The AsciiArt theme supports three primary layouts:

  • No Sidebar Demo
  • Sidebar Full – Sidebar with header upon aside. Demo
  • Sidebar upon content Demo

No sidebar

This is the most basic layout containing a basic header,navigation and content. No bells no whistles.

Sidebar Full Layout

A page containing sidebar taking over the whole page height.
The main navigation rests next to sidebar whereas the content rests underenath it.

Upon small screen or upon window resizing sidebar moves offcanvas allowing the user to focus on content.

To use this layout, first include the theme's CSS:

CSS

<link rel="stylesheet" href="css/style.css">

Then include the JavaScript file and initialize the Sidebar. Note that the JavaScript does not initialize automatically—you must explicitly create a new Sidebar instance:

JavaScript

<script type="application/javascript" src="/js/asciiart.min.js"></script>
<script type="application/javascript">
    document.addEventListener("DOMContentLoaded", () => {
        // Initialize the Sidebar component
        new AsciiartTheme.Sidebar("#sidebar");
    });
</script>

Alternatively, you can use an ES6 module to import and initialize the Sidebar component:

ES6 Module

<script type="component">
    import { Sidebar } from "js/asciiart.esm.min.js";

    document.addEventListener("DOMContentLoaded", () => {
        // Initialize the Sidebar using ES6 module
        new Sidebar("#sidebar");
    });
</script>

Sidebar upon content

An extension of no-sidebar layout is to place underneath the main navigation a sidebar.
This is useful for placing a secondary navigation upon main content.

When browser window shinks it becomes hidden allowing you to focus upon main content.
