Setup

Javascript

About

Javascript is crucial for initializing the components described upon components page. Here we showcase how can be used.

Classic Usage

Using asciiart.min.js you can use it as traditional Javascript module:

Html

<script type="application/javascript" src="/js/asciiart.min.js></script>

Browser Module Usage

Alternatively you can use the esm build asciiart.esm.min.js in order to embed code as browser module:

Html

<script type="application/javascript" src="/js/asciiart.min.js></script>
<script type=\"module\">
    import { Sidebar} from "/js/asciiart.esm.js"; 
    document.addEventListener("DOMContentLoaded",function(){
        new Sidebar("#sidebar")
        //Initialize components here
    })
</script>
