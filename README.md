# AsciiArt

A fully responsive, customizable HTML/CSS theme inspired by the charm of 7-bit ASCII art.

## Install:

```
npm install @pc_magas/asciiart
```

## Minified CSS and Js files:

Upon `node_modules/dist` both minified JS and CSS files exist:

File | Usage
--- | ---
@pc_magas/asciiart/dist/js/asciiart.min.js | Classic Js file you can copy it and use it as traditional javascript 
@pc_magas/asciiart/dist/js/asciiart.ems.min.js | Javascript file to use as module into you js code
@pc_magas/asciiart/dist/css/style.css | Css file

## Build from source

```
git clone git@gitlab.com:pc-magas/asciiart-html-theme.git
cd ./asciiart-html-theme
npm build
```
